# 在监狱的墙上开一扇窗

​2014年春天，艺术家阿尔弗雷德·埃斯皮诺萨与“监狱之窗”项目组以及艾滋病宣传组织“费城FIGHT”和监狱健康报共同合作，提供一系列空白明信片给在押人员。向他们征询：

“如果你能在监狱里建一扇窗户，你希望这个世界看到什么？”

鼓励在押人员以画作及文字形式反馈。

搜集完成后，这些明信片中的一部分被印制出来面向公众，同时寄给监狱。
