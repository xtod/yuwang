# 欧飞鸿与《沙湾501》

​本期狱望为大家带来的是欧飞鸿和他的《沙湾501》。

![p8.23.1](./images/8.23.1.jpg)

2016年底，因为涂鸦，欧飞鸿被控寻衅滋事，在广州番禺沙湾镇看守所501监仓度过了7个月的时光。

![p8.23.2](./images/8.23.2.jpg)

期间，他画了十多本册子，其中一部分得以陆续寄出。去年，他将这段经历以及部分画装订成册，也就是今天介绍的《沙湾501》。
