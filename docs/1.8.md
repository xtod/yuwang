# 人  间  

为了了解监狱，我去一个叫坐牢的贴吧逛了逛。以我不多的贴吧经验，这个吧最让我感动。发言的大部分是守候自己老公或者男朋友的女性家属。不少人每天到贴吧打卡，为自己男人回家倒计时，并写下一些什么，比如：还有1845天你就回来了；老公，你离开已经43天，我好想你之类的。稍微浏览了一下后，我进了一个犯人家属群，里面40来人，大部分是女性。看了不到一个星期，心有不忍，想退群。抄下部分发言，按聊天体贴在下面：
